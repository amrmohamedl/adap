## 📦 What is adap?

adap stands for *Another Day Another Package*. It is a simple tool designed to help you manage packages without any coding work. Whether you use packages for updating software or handling code libraries, adap helps you do it with minimal effort.

This app targets Windows users who want an easier way to organize and update packages. You do not need to know anything about programming.

## 📋 Common Troubleshooting

**If adap does not start:**

- Make sure your Windows version is supported.

- Restart your computer and try again.

- Check if antivirus or firewall is blocking the app.

**If package download fails:**

- Check your internet connection.

- Restart adap and try downloading again.

- Wait and try later if servers are busy.

**If installation fails midway:**

- Close adap and all other running programs.

- Run the installer or adap as an administrator by right-clicking the icon and choosing "Run as administrator".

## 📂 Where does adap save files?

By default, adap stores its settings and downloaded packages in:

You can change this folder inside adap’s settings if needed.
